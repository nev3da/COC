# 部落冲突打资源脚本

只能打资源，支持主世界和夜世界<br>
不了解python可以在右侧Releases下载main.zip，里面有exe和使用说明<br>
不支持多显示器<br>
运行后会抢鼠标，推荐在不使用电脑的时候运行<br>

1. pip install -r requirements.txt
2. 使用 MuMu 模拟器（其他模拟器没试过）
3. 设置为平板模式，分辨率 1600x900
4. （如果设为平板模式失败）机型设置 --> 自定义型号
   - 手机品牌：iPad
   - 手机型号：Air5
   - 入网型号：NB7095
5. 重启模拟器

## 夜世界

1. 进入夜世界，所有兵营全部放一个兵种（最好是女巫）
2. 以管理员身份运行python main.py
3. 运行后不要移动游戏窗口

## 主世界

1. 进入主世界（解锁了飞龙和蝙蝠法术）
2. 以管理员身份运行python main.py
3. 运行后不要移动游戏窗口
4. 推荐装备：
   - 女王：穿云箭+天使
   - 蛮王：灵蛇+足球
   - 大守护者：狂暴+金身
   - 闰土：随意
   - 亡灵王子：减速法球+随意
