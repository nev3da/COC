# 部落冲突打资源脚本

只能打资源，支持主世界和夜世界<br>
不了解编程可以在右侧Releases下载main.zip，里面有exe和使用说明<br>
没有主动适配多显示器（可能支持，但我没试过）<br>
可后台运行，不抢鼠标，运行后可以用电脑干别的事情，但**不要把模拟器窗口最小化**<br>
点击停止后不会立刻停止运行，而是在运行到一些检查点时才停止

1. pip install -r requirements.txt
2. 使用 MuMu 模拟器（其他模拟器没试过，应该也行）
3. 设置为平板模式，分辨率 **1600x900**（**分辨率一定要对**）
4. （如果设为平板模式失败）机型设置 --> 自定义型号
   - 手机品牌：iPad
   - 手机型号：Air5
   - 入网型号：NB7095
5. 重启模拟器

## 夜世界

1. 进入夜世界，所有兵营全部放一个兵种（最好是女巫）
2. 以管理员身份运行python main.py

## 主世界

1. 进入主世界（解锁了飞龙和蝙蝠法术）
2. 以管理员身份运行python main.py
3. 推荐装备：
   - 女王：穿云箭+天使
   - 蛮王：灵蛇+足球
   - 大守护者：狂暴+金身
   - 闰土：随意
   - 亡灵王子：减速法球+随意
